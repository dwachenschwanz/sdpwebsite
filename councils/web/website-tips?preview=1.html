Login prompt for forms

in the link address, add "login.php?ret_link=/" in between the domain and the form: "www.decisionprofessionals.com/login.php?ret_link=/forms/abcdefg9999

 if a non-logged-in user follows a link to a web page associated with membership groups (ie behind the paywall), the system automatically sends them to the login page, however, for forms this auto-redirect does not occur, and hence needs to be explicitly hard-coded
